Как заменить файлы в репозитории:

1. Удали старые файлы из проекта, кроме .git:
   find . -mindepth 1 -maxdepth 1 ! -name '.git' -exec rm -rf {} +

2. Распакуй содержимое этого архива в корень репозитория.

3. Установи зависимости:
   npm install

4. Запуск локально:
   npm run dev

5. Сборка:
   npm run build

Дополнительно:
- Положи loader.mp4 в корень проекта для видео загрузки.
- Положи track.mp3 в корень проекта для кнопки Music.
